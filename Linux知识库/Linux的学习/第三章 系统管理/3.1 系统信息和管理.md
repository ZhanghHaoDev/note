## 2. 关机和重启

### 关机命令

- **shutdown**：计划关机或重启。
  ```bash
  shutdown [选项] [时间] [消息]
    ```
shutdown的选项：
+ -h：关机。
+ -r：重启。
+ -c：取消计划的关机或重启。
+ -P：关机并关闭电源。

```bash
shutdown -h now          # 立即关机
shutdown -h +10 "即将关机"  # 10分钟后关机，并显示消息
shutdown -r now          # 立即重启
shutdown -c              # 取消计划的关机或重启
```

### 重启命令
