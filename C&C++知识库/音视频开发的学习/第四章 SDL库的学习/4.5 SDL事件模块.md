# SDL事件模块

SDL事件模块
SDL事件模块：提供了事件处理的功能，包括键盘、鼠标、窗口事件等。
SDL_PollEvent(): 轮询事件。
SDL_WaitEvent(): 等待事件。
SDL_Event: 事件结构体，包含各种事件类型的信息。
