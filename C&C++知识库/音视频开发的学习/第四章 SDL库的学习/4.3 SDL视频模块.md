# SDL 视频模块

1. 代码路径：sdl_stu/sdl_video_module
2. 测试代码：sdl_stu/test/sdl_video_module_test.cc
3. sdl没有提供播放视频的接口，需要ffmpeg库配合使用才可以播放

## 1. SDL视频模块简介

SDL视频模块提供了创建窗口、渲染图形和处理视频的功能。它支持2D图形渲染，并可以与OpenGL结合使用进行3D图形渲染。

初始化SDL，需要使用`SDL_Init(SDL_INIT_VIDEO)`

## 2. 创建和管理窗口

SDL（Simple DirectMedia Layer）视频模块提供了强大的窗口创建和管理功能，适用于跨平台的多媒体应用和游戏开发。

1. 创建窗口

函数原型：
```cpp
SDL_Window* SDL_CreateWindow(const char* title, int x, int y, int w, int h, Uint32 flags);
```
函数返回值：返回创建的窗口指针，失败时返回NULL。

函数参数：
+ title：窗口标题。
+ x：窗口的x坐标。
+ y：窗口的y坐标。
+ w：窗口的宽度。
+ h：窗口的高度。
+ flags：窗口的属性标志，如SDL_WINDOW_SHOWN表示显示窗口。

2. 窗口大小调整

函数原型：
```cpp
void SDL_SetWindowSize(SDL_Window* window, int w, int h);
```

函数返回值：无返回值

函数参数：
+ window：窗口指针。
+ w：新的宽度。
+ h：新的高度。

3. 窗口标题设置

函数原型：
```cpp
void SDL_SetWindowTitle(SDL_Window* window, const char* title);
```
函数返回值：无返回值

函数参数：
+ window：窗口指针。
+ title：新的窗口标题。

4. 窗口全屏模式

函数原型：
```cpp
int SDL_SetWindowFullscreen(SDL_Window* window, Uint32 flags);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ window：窗口指针。
+ flags：全屏模式标志，如SDL_WINDOW_FULLSCREEN。

5. 获取窗口信息，窗口位置

函数原型：
```cpp
void SDL_GetWindowSize(SDL_Window* window, int* w, int* h);
```

函数返回值：无返回值。

函数参数：
+ window：窗口指针。
+ x：指向存储x坐标的指针。
+ y：指向存储y坐标的指针。

6. 设置窗口大小

函数原型：
```cpp
void SDL_SetWindowSize(SDL_Window* window, int w, int h);
```

函数返回值：无返回值。

函数参数：
+ window：窗口指针。
+ w：新的宽度。
+ h：新的高度。

7. 销毁窗口

函数原型：
```cpp
void SDL_DestroyWindow(SDL_Window* window);
```

函数返回值：无返回值。

函数参数：window：窗口指针。

## 3. 渲染和绘制

SDL（Simple DirectMedia Layer）视频模块提供了强大的渲染和绘制功能，适用于跨平台的多媒体应用和游戏开发。以下是SDL在渲染和绘制方面的主要功能和使用示例。

1. 创建渲染器

```cpp
SDL_Renderer* SDL_CreateRenderer(SDL_Window* window, int index, Uint32 flags);
```
函数返回值：返回创建的渲染器指针，失败时返回NULL。

函数参数：
+ window：窗口指针。
+ index：渲染驱动程序索引。
+ flags：渲染器的属性标志，如SDL_RENDERER_ACCELERATED表示使用硬件加速。

2. 设置渲染颜色

函数原型：
```cpp
int SDL_SetRenderDrawColor(SDL_Renderer* renderer, Uint8 r, Uint8 g, Uint8 b, Uint8 a);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ renderer：渲染器指针。
+ r：红色值。
+ g：绿色值。
+ b：蓝色值。
+ a：透明度值。

3. 清除渲染目标

函数原型：
```cpp
int SDL_RenderClear(SDL_Renderer* renderer);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：renderer：渲染器指针。

4. 绘制纹理

函数原型：
```cpp
int SDL_RenderCopy(SDL_Renderer* renderer, SDL_Texture* texture, const SDL_Rect* srcrect, const SDL_Rect* dstrect);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ renderer：渲染器指针。
+ texture：纹理指针。
+ srcrect：源矩形。
+ dstrect：目标矩形

5. 显示渲染结果
```cpp
void SDL_RenderPresent(SDL_Renderer* renderer);
```

函数返回值：无返回值

函数参数：renderer：渲染器指针。

6. 销毁渲染器
```cpp
void SDL_DestroyRenderer(SDL_Renderer* renderer);
```

函数返回值：无返回值。

函数参数：renderer：渲染器指针。

7. 绘制矩形
```cpp
int SDL_RenderDrawRect(SDL_Renderer* renderer, const SDL_Rect* rect);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ renderer：渲染器指针。
+ rect：指向要绘制的矩形的指针。

8. 填充矩形
```cpp
int SDL_RenderFillRect(SDL_Renderer* renderer, const SDL_Rect* rect);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ renderer：渲染器指针。
+ rect：指向要填充的矩形的指针。

9. 绘制线条
```cpp
int SDL_RenderDrawLine(SDL_Renderer* renderer, int x1, int y1, int x2, int y2);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ renderer：渲染器指针。
+ x1：起点的x坐标。
+ y1：起点的y坐标。
+ x2：终点的x坐标。
+ y2：终点的y坐标。

10. 绘制点

函数原型：
```cpp
int SDL_RenderDrawPoint(SDL_Renderer* renderer, int x, int y);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ renderer：渲染器指针。
+ x：点的x坐标。
+ y：点的y坐标。

## 4. 处理图像

SDL（Simple DirectMedia Layer）视频模块提供了丰富的图像处理功能，适用于跨平台的多媒体应用和游戏开发。以下是SDL在处理图像方面的主要功能和使用示例。

需要包含头文件：`#include "SDL2/SDL_image.h"`
需要安装sdl的image库：`brew install sdl2_image`

1. 加载图像

函数原型：
```cpp
SDL_Surface* IMG_Load(const char* file);
```

函数返回值：返回加载的表面指针，失败时返回NULL。

函数参数：file：图像文件路径。

2. 创建纹理

函数原型：
```cpp
SDL_Texture* SDL_CreateTextureFromSurface(SDL_Renderer* renderer, SDL_Surface* surface);
```

函数返回值：返回创建的纹理指针，失败时返回NULL。

函数参数：
+ renderer：渲染器指针。
+ surface：表面指针。

3. 释放表面

函数原型：
```cpp
void SDL_FreeSurface(SDL_Surface* surface);
```

函数返回值：无返回值

函数参数：surface：表面指针。

4. 销毁纹理

函数原型：
```cpp
void SDL_DestroyTexture(SDL_Texture* texture);
```

函数返回值：无返回值

函数参数：texture：纹理指针。

6. 图像格式转换

函数原型：
```cpp
SDL_Surface* SDL_ConvertSurfaceFormat(SDL_Surface* src, Uint32 pixel_format, Uint32 flags);
```

函数返回值：返回转换后的表面指针，失败时返回NULL。

函数参数：
+ src：源表面指针。
+ pixel_format：目标像素格式。
+ flags：转换标志。

7. 图像缩放

函数原型：
```cpp
int zoomSurfaceRGBA(SDL_Surface* src, SDL_Surface* dst, double zoomx, double zoomy, int smooth);
```

函数返回值：返回0表示成功，返回负值表示失败。

函数参数：
+ src：源表面指针。
+ dst：目标表面指针。
+ zoomx：水平缩放因子。
+ zoomy：垂直缩放因子。
+ smooth：是否使用平滑缩放。

8. 图像旋转

函数原型：
```cpp
SDL_Surface* rotozoomSurface(SDL_Surface* src, double angle, double zoom, int smooth);
```

函数返回值：返回旋转后的表面指针，失败时返回NULL。

函数参数：
+ src：源表面指针。
+ angle：旋转角度。
+ zoom：缩放因子。
+ smooth：是否使用平滑缩放。
