# SDL音频模块

SDL音频模块：提供了音频设备的管理和音频数据的播放功能。
SDL_OpenAudio(): 打开音频设备。
SDL_CloseAudio(): 关闭音频设备。
SDL_PauseAudio(): 暂停或恢复音频播放。
SDL_LoadWAV(): 加载WAV音频文件。
