# SDL线程模块

SDL线程模块：提供了创建和管理线程的功能。
SDL_CreateThread(): 创建一个线程。
SDL_WaitThread(): 等待线程结束。
SDL_LockMutex(): 锁定互斥量。
SDL_UnlockMutex(): 解锁互斥量。
