# UNIX网络编程的学习

## 1. UNIX网络编程的简介

1. 网络开发是指通过网络技术，实现不同计算机之间的数据交换和通信，使得计算机之间能够相互协作，共享资源，提高工作效率。
2. 代码仓库：unix_stu/unix_network_stu
3. 测试仓库：unix_stu/test/unix_network_stu
4. 开发环境：操作系统：macOS，编辑器：VSCode，构建工具：CMake，编译器：llvm
