# 线程（Threads）

## 1. 线程 简介

### 线程 note

1. 线程：线程是操作系统能够进行运算调度的最小单位。它被包含在进程之中，是进程中的实际运作单位。一条线程指的是进程中一个单一顺序的控制流。一个进程可以并发多个线程，每个线程并行执行不同的任务，但它们共享相同的进程资源，如内存空间、文件描述符等。

2. 线程和进程的区别：在Linux下，线程是最小的执行单位，进程是最小的资源分配单位。进程适用于高可靠和隔离性要求的场景，线程适用于高并发和高性能要求的场景。

3. 线程的优点：
    - 线程之间的切换开销比进程小。
    - 线程可以共享进程的资源，减少资源占用。
    - 线程间通信更高效，因为它们共享相同的地址空间。

4. 线程的缺点：
    - 线程之间共享进程资源，可能导致资源竞争问题。
    - 线程的编程复杂度较高，需要处理同步和互斥问题。
    - 一个线程崩溃可能导致整个进程崩溃。

5. 线程的应用场景：
    - 多线程服务器：如Web服务器，可以使用线程处理多个客户端请求。
    - 并行计算：将计算任务分解为多个线程并行执行，提高计算效率。
    - 实时系统：如操作系统内核，使用线程处理不同的实时任务。

## 2. 线程 接口函数

1. 线程创建
+ 函数功能：创建一个新的线程
+ 函数原型：
    ```c
    #include <pthread.h>
    int pthread_create(pthread_t *thread, const pthread_attr_t *attr, void *(*start_routine)(void *), void *arg);
    ```
+ 函数参数：
  - `thread`：指向线程标识符的指针
  - `attr`：线程属性，传递NULL使用默认属性
  - `start_routine`：线程运行函数的起始地址
  - `arg`：传递给线程运行函数的参数
+ 函数返回值：成功返回0，失败返回错误码

2. 线程终止
+ 函数功能：终止调用线程
+ 函数原型：
    ```c
    #include <pthread.h>
    void pthread_exit(void *retval);
    ```
+ 函数参数：
  - `retval`：线程的返回值
+ 函数返回值：无

3. 线程等待
+ 函数功能：等待指定线程终止
+ 函数原型：
    ```c
    #include <pthread.h>
    int pthread_join(pthread_t thread, void **retval);
    ```
+ 函数参数：
  - `thread`：要等待的线程标识符
  - `retval`：指向线程返回值的指针
+ 函数返回值：成功返回0，失败返回错误码

4. 线程分离
+ 函数功能：将线程设置为分离状态
+ 函数原型：
    ```c
    #include <pthread.h>
    int pthread_detach(pthread_t thread);
    ```
+ 函数参数：
  - `thread`：要分离的线程标识符
+ 函数返回值：成功返回0，失败返回错误码

5. 线程取消
+ 函数功能：请求取消目标线程
+ 函数原型：
    ```c
    #include <pthread.h>
    int pthread_cancel(pthread_t thread);
    ```
+ 函数参数：
  - `thread`：要取消的线程标识符
+ 函数返回值：成功返回0，失败返回错误码

6. 线程ID，标识符
+ 函数功能：获取调用线程的线程标识符
+ 函数原型：
    ```c
    #include <pthread.h>
    pthread_t pthread_self(void);
    ```
+ 函数参数：无
+ 函数返回值：返回调用线程的线程标识符

### 线程 同步

1. 互斥锁
+ 函数功能：初始化互斥锁
+ 函数原型：
    ```c
    #include <pthread.h>
    int pthread_mutex_init(pthread_mutex_t *mutex, const pthread_mutexattr_t *attr);
    ```
+ 函数参数：
  - `mutex`：指向互斥锁对象的指针
  - `attr`：互斥锁属性，传递NULL使用默认属性
+ 函数返回值：成功返回0，失败返回错误码

+ 函数功能：销毁互斥锁
+ 函数原型：
    ```c
    #include <pthread.h>
    int pthread_mutex_destroy(pthread_mutex_t *mutex);
    ```
+ 函数参数：
  - `mutex`：指向互斥锁对象的指针
+ 函数返回值：成功返回0，失败返回错误码

+ 函数功能：加锁互斥锁
+ 函数原型：
    ```c
    #include <pthread.h>
    int pthread_mutex_lock(pthread_mutex_t *mutex);
    ```
+ 函数参数：
  - `mutex`：指向互斥锁对象的指针
+ 函数返回值：成功返回0，失败返回错误码

+ 函数功能：解锁互斥锁
+ 函数原型：
    ```c
    #include <pthread.h>
    ```

+ 函数参数：

2. 条件变量
+ 函数功能：初始化条件变量
+ 函数原型：

+ 函数参数：
